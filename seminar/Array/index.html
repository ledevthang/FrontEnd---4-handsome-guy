Cách khai báo mảng

Có 2 cách khai báo mảng là :

Cách 1. Khai báo bằng dấu []

var array = ["cho","meo","ca"]

Cách 2. Sử dụng từ khóa new Array():

var array = new Array();

Cả 2 cách đều giống nhau, nhưng nên sử dụng cách 1(array literal method).

Tham chiếu đến phần tử trong mảng

Có rất nhiều cách tham chiếu đến phần tử trong mảng như:

Gán:

var name = thang[0];

Sửa giá trị:

thang[0] = "xau trai";

Tham chiếu đến cả array

var meo = ["meo con","meo me","meo bo"];
var cho = meo;

Cách 2. Sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo:

var Conmeo = {
    duoi : "",
    tai : "",
    mat : "",
    long : ""
};

Cách 3. Sử dụng từ khóa {} và thêm thuộc tính ngay sau đó:

var Conmeo = {};
Conmeo.duoi : "",;
Conmeo.tai : "",;
Conmeo.mat : "",;
Conmeo.long : "";

Array vs Object

Array là một kiểu đặc biệt của Object.

Toán tử typeof sẽ trả về "object" cho array

Array sử dụng số để tham chiếu đến các element:

var cars = ["banh xe","vo lang"]; VD: cars[0];

Còn object thì sử dụng name để tham chiếu:

var a = {ten: 'thang'};
a.ten;

Phan tu cua array

Bạn có thể có những biến khác nhau trên cùng 1 array như Object, function hay cả array:

var array = [
array[0] = Date.now;
array[1] = myFunction;
array[2] = myCars;
]

Length property

Length của array luôn luôn lớn hơn số lớn nhất của array index:

var a = ["1","2","3"];
a.length ==> //3 (array index thì sẽ là 2);

Array methods

Một số method hay dùng của array:

Hàm toString convert array sang String:
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.toString();
==> Banana,Orange,Apple,Mango;

Hàm join() cũng sẽ trả từ array về string:
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.join(" * ");
==> Banana * Orange * Apple * Mango;

Hàm pop()
Remove phần tử cuối cùng của array và return phần tử bị remove đó.

Hàm push()
Add 1 phần tử vào vị trí cuối cùng của array va return array length

Hàm shift()
Remove phần tử đầu tiên của array và return phần tử remove

Hàm unshift()
Add phần tử vào vị trí đầu tiên của array va return phần tử bị remove

Hàm delete
Xóa 1 phần tử và trả về undefined.
//phần tử vẫn sẽ xuất hiện tại array nhưng với value là undefined.

Hàm splice()
Remove phần tử và trả về array
var a = ["1","2","3","4"];
a.splice(2,1) ==> //["1","2","4"];
2 : là vị trí bắt đầu;
1 : là số phần tử remove;

Hàm concat()
Merge 2 array
var array1 = ["1","2","3"];
var array2 = ["4","5","6"];
var array3 = ["7","8","9"];


merge array
var merge = array1.concat(array2);
==> return merge = ["1","2","3","4","5","6"];


marge 3 array var merge2 = array1.concat(array2,array3);
==> return merge2 = ["1","2","3","4","5","6","7","8","9"];


merge value var merge3 = array1.concat(["10"]);
==> return merge2 = ["1","2","3","10"];

Hàm slice()
"Thái" các phần tử bắt đầu từ vị trí được chọn và trả về array mới:
var a = ["1","2","3","4"];
var x = a.slice(1) ==> cắt từ vị trí index 1, return x = [2","3","4"];

Thay đổi phần tử:

var a = ["phantu1","phantu2","phantu3"]; a[0] = "thayphantu1"; //array index bắt đầu từ 0;
