Cach khai bao mang

Co 2 cach khai bao mang la :

Cach 1. Khai bao bang dau []

var array = ["thang","dep","trai"]

Cach 2. Sử dụng từ khóa new Array():

var array = new Array();

Ca 2 cach deu giong nhau, nhung nen su dung cach 1 (array literal method).

Tham chieu den phan tu trong mang

Co rat nhieu cach tham chieu den phan tu trong mang nhu:

Gán:

var name = thang[0];

Sửa giá trị:

thang[0] = "xau trai";

Tham chiếu đến cả array

var meo = ["meo con","meo me","meo bo"];
var cho = meo;

Cách 2. Sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo:

var Conmeo = {
    duoi : "",
    tai : "",
    mat : "",
    long : ""
};

Cách 3. Sử dụng từ khóa {} và thêm thuộc tính ngay sau đó:

var Conmeo = {};
Conmeo.duoi : "",;
Conmeo.tai : "",;
Conmeo.mat : "",;
Conmeo.long : "";

Array vs Object

Array là một kiểu đặc biệt của Object.

Toán tử typeof sẽ trả về "object" cho array

Array sử dụng số để tham chiếu đến các element:

var cars = ["banh xe","vo lang"]; VD: cars[0];

Còn object thì sử dụng name để tham chiếu:

var a = {ten: 'thang'};
a.ten;

Phan tu cua array

Bạn có thể có những biến khác nhau trên cùng 1 array như Object, function hay cả array:

var array = [
array[0] = Date.now;
array[1] = myFunction;
array[2] = myCars;
]

Length property

Length cua array luon luon lon hon so lon nhat cua array index:

var a = ["1","2","3"];
a.length ==> //3 (array index thi se la 2);

Array methods

Mot so method hay dung cua array:

Ham toString convert array sang String:
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.toString();
==> Banana,Orange,Apple,Mango;

Ham join() cung se tra array ve string:
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.join(" * ");
==> Banana * Orange * Apple * Mango;

Ham pop()
Remove phan tu cuoi cua array va return phan tu bi remove

Ham push()
Add 1 phan tu vao vi tri cuoi cua array va return array length

Ham shift()
Remove phan tu dau tien cua array va return phan tu bi remove

Ham unshift()
Add phan tu vao vi tri dau tien cua array va return phan tu bi remove

Ham delete
Xoa 1 phan tu va tra ve undefined.
//phan tu se van xuat hien tai array nhung voi value la undefined.

Ham splice()
Remove phan tu va tra ve array
var a = ["1","2","3","4"];
a.splice(2,1) ==> //["1","2","4"];
2 : là vị trí bắt đầu;
1 : là số phần tử remove;

Ham concat()
Merge 2 array
var array1 = ["1","2","3"];
var array2 = ["4","5","6"];
var array3 = ["7","8","9"];


merge array
var merge = array1.concat(array2);
==> return merge = ["1","2","3","4","5","6"];


marge 3 array var merge2 = array1.concat(array2,array3);
==> return merge2 = ["1","2","3","4","5","6","7","8","9"];


merge value var merge3 = array1.concat(["10"]);
==> return merge2 = ["1","2","3","10"];

Ham splice()
"Thái" cac phan tu bat dau tu vi tri duoc chon va se tra ve array moi:
var a = ["1","2","3","4"];
var x = a.slice(1) ==> cat tu vi tri index 1, return x = [2","3","4"];

Thay đổi phần tử:

var a = ["phantu1","phantu2","phantu3"]; a[0] = "thayphantu1"; //array index bắt đầu từ 0;
