Cách khởi tạo object

Ta có 5 kiểu dữ liệu cơ bản và 1 kiểu dữ liệu phức hợp. 5 kiểu dữ liệu cơ bản bao gồm: Number, String, Boolean, Undefined và Null. Kiểu dữ liệu phức hợp là kiểu dữ liệu Object.

Một đối tượng (một object) là một danh sách các item, mỗi item là một cặp name-value, trong đó value có thể là: các kiểu dữ liệu cơ bản, function, hay cũng có thể là một object khác (kiểu dữ liệu phức hợp).

Có 2 cách để khởi tạo:

Cach 1. Sử dụng từ khóa new Object():

var obj = new Object();

Cach 2. Sử dụng từ khóa {}:

var obj = {};

Tạo thuộc tính

Thuộc tính là những đặc điểm (có thể hiểu là biến) cần lưu trữ trong đối tượng.

Ta co the khai bao bang 3 cach

Cách 1. Sử dụng từ khóa new Object():

var Conmeo = new Object();
// Tạo thuộc tính
Conmeo.duoi = '';
Conmeo.tai = '';
Conmeo.mat = '';
Conmeo.long = '';

Cách 2. Sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo:

var Conmeo = {
    duoi : "",
    tai : "",
    mat : "",
    long : ""
};

Cách 3. Sử dụng từ khóa {} và thêm thuộc tính ngay sau đó:

var Conmeo = {};
Conmeo.duoi : "",;
Conmeo.tai : "",;
Conmeo.mat : "",;
Conmeo.long : "";

Từ khóa this

Từ khóa this dùng để đại diện cho 1 đối tượng

var Conmeo = {};
Conmeo.duoi : "dài",;
Conmeo.tai : "to",;
Conmeo.mat : "xanh",;
Conmeo.long : "mượt";
console.log('this.tai + " rat la thinh "')

- Sau một thời gian phát triển thì file JS của bạn sẽ rất lớn và rất khó kiểm sát được các biến,để tránh bị nhầm lẫn chúng ta nên sử dụng từ khóa this

- Từ khóa this của JavaScript được sử dụng để tham chiếu đến các đối tượng mà hàm (nơi mà this được sử dụng) bị ràng buộc.

- Từ khóa this không chỉ tham chiếu đến đối tượng, mà nó còn chứa giá trị của đối tượng đó

Truy cập và gán giá trị

Có thể truy cập đến thuộc tính của Object theo 2 cách:

tenObject.tenThuoctinh

hoặc

tenObject["tenThuoctinh"]

Gán giá trị

tenObject.tenThuoctinh = "something"

hoặc dùng this trong function

var Comment = {
title : "",
addComment : function(){
this.title = "Tiêu đề bình luận"; } };

Lấy giá trị cho biến

var duoi = Conmeo.duoi;

hoặc dùng this

var Comment = {
title : "",
addComment : function(){
var title = this.title; } };

Object Methods

Object cũng có methods
Methods là các hành động có thể được thực hiện trên các Object.
Nó có thể được đặt trong thuộc tính như các function.

ví dụ về methods trong Object

var person = {
firstName: "John",
lastName : "Doe",
id : 5566,
fullName : function() {
return this.firstName + " " +this.lastName;
} };
