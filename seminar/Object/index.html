Cách khởi tạo object

Co 2 cach de khoi tao:

Cach 1. Su dung tu khoa new Object():

var obj = new Object();

Cach 2. Su dung tu khoa {}:

var obj = {};

Tạo thuộc tính

Thuộc tính là những đặc điểm (có thể hiểu là biến) cần lưu trữ trong đối tượng.

Ta co the khai bao bang 3 cach

Cách 1. Sử dụng từ khóa new Object():

var Conmeo = new Object();
// Tạo thuộc tính
Conmeo.duoi = '';
Conmeo.tai = '';
Conmeo.mat = '';
Conmeo.long = '';

Cách 2. Sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo:

var Conmeo = {
    duoi : "",
    tai : "",
    mat : "",
    long : ""
};

Cách 3. Sử dụng từ khóa {} và thêm thuộc tính ngay sau do:

var Conmeo = {};
Conmeo.duoi : "",;
Conmeo.tai : "",;
Conmeo.mat : "",;
Conmeo.long : "";
