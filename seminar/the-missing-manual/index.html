Trong Css:

+ Inline Style ưu tiên cao nhất
+ Sau đó là Id
+ Tiếp theo là Class
+ Và cuối cùng là Element
+ Trong Css phần tử con có thể kế thừa lại thuộc tính của phần tử cha như các thuộc tính font-family, font-size, color...
+ Hoặc chúng ta cũng có thể dùng thuộc tính inherit.
+ Format text bằng thuộc tính css như: font-family, font-size, font-style, font-weight,font-variant,color,text-shadow...
+ Format text bằng tag như: strong, em, b,i...
+ Margin: khoảng cách từ border của phần tử với phần tử gần kề(không thay đổi kích thước khi set margin).
+ Padding: khoảng cách từ nội dung bên trong phần tử với border của nó(sẽ làm tăng kích thước phần tử).
+ Chúng ta có thể set margin/padding 4 vị trí là top, right, bottom, left.
+ Thuộc tính border dùng để định dạng đường viền của phần tử.
+ Có 3 loại thuộc tính chính là border-style,border-width,border-color
+ div { border: 1px(border-width) double(border-style) red(border-color) }
+ Responsive là kĩ thuật thiết kế web phù hợp với nhiều loại thiết bị màn hình và đều hiển thị tốt,không bị vỡ layout.
+ Để responsive web chúng ta có thể dùng Media Query hoặc Bootstrap, Flexbox cùng với đó là không nên fix cứng width và height.
+ Media Query giúp website có thể hiện thị tốt ở tất cả các loại màn hình khác nhau như điện thoại,PC,Laptop
+ Cấu trúc @media media_type {code}
